## Install
### Install package in direcory client
  ```
  cd client
  cp env-example .env
  yarn
  ```
### Install package in direcory server  
  ``` 
  cd server
  cp env-example .env
  yarn
  ```
  
## Get started
  ```
  cd server
  yarn start
  ```
  ```
  cd client
  yarn start
  ```
